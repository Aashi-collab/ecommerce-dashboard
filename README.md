Objective

The main objective of this project is to analyze e-commerce sales data and uncover insights related to revenue, customer trends, product performance, and regional sales distribution.

Dashboard Features
Interactive dashboard with filters and slicers
Sales performance overview (Total Sales, Profit, Orders)
Category-wise and product-wise analysis
Region-wise and state-wise sales distribution
Monthly and yearly sales trends
Top-selling products identification
Customer segment analysis

Key Insights
Certain product categories contribute significantly to total revenue
Sales show seasonal trends with peak periods
Some regions generate higher sales compared to others
A small number of products drive a large portion of revenue (Pareto principle)
Customer segmentation helps identify high-value customers

Tools & Technologies
Visualization Tool: Power BI
Data Source: CSV / Excel dataset
Data Cleaning: Power Query
Data Modeling: Relationships, DAX measures

Dataset Details

The dataset includes key attributes such as:

Order ID
Order Date
Customer Name / Segment
Product Category & Sub-category
Sales Amount
Profit
Quantity
Region / State
